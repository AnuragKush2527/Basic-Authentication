Project Name: Basic Authentication

A full-stack web application built with the MERN stack (MongoDB, Express, React, Node.js).

Prerequisites:

Node.js
npm or yarn
MongoDB (Local or Atlas)

Installation & Setup:

1. Clone the repository:

git clone https://github.com/AnuragKush2527/Basic-Authentication.git
cd Basic-Authentication

2. Backend Setup:

cd backend
npm install

3. Start the backend:

npm run dev

4. Frontend Setup:

cd ../frontend
npm install

5. Start the frontend:

npm run dev
